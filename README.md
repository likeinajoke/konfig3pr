# Практика 4. Ассемблер УВМ

## Этап 1

**Цель этапа:**  
Перевести текстовую программу на простом ассемблерном языке в *промежуточное представление (IR)*.

IR — это список инструкций, каждая представлена словарём:

- `mnemonic` — имя команды
- `A` — код инструкции
- `B`, `C`, `D`, `CONST` — аргументы инструкции (если есть)

---

## Синтаксис языка ассемблера

#### **1. Присваивание константы**
```asm
a = 5 # переменная → адрес в памяти
mem[500] = 42 # запись константы в память
```
Эквивалентно: `LOAD_CONST`

#### **2. Чтение из памяти**
```asm
x = mem[a]
```
Эквивалентно: `READ`

#### **3. Запись в память**
```asm
mem[b] = x
```
Эквивалентно: `WRITE`

#### **4. Операция min()**
```asm
z = min(x, y)
mem[493] = min(mem[333], mem[1003])
```
Эквивалентно: `MIN`
